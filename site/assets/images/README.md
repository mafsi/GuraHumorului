# Photos go here

Every `.gh-placeholder` block on the site shows a caption describing what
photo belongs there. Drop real photos into this folder (JPEG or WebP,
reasonably compressed for web) using the suggested filenames below, then
swap each placeholder `<div class="gh-placeholder">...</div>` for an
`<img src="assets/images/<filename>" alt="...">` (or `../assets/images/...`
from an `en/` page).

Same photo, reused across pages, only needs to be added once.
